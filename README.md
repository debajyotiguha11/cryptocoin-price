### Installation

```
git clone https://github.com/techstar-inc/cryptocoin-price.git

cd cryptocoin-price

sudo apt-get install python-gi
sudo apt-get install python-requests
sudo apt-get install gir1.2-appindicator3-0.1

python indicator.py 
```

### Or 
```
make install
make run
```
